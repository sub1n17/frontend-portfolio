# Subin Portfolio

Next.js와 TypeScript 기반으로 구현한 개인 포트폴리오 웹사이트입니다.

## 🔗 Links

- Deploy: https://choisubin.vercel.app
- GitHub: https://github.com/sub1n17/frontend-portfolio

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- CSS Module

## 📄 Pages

- **Main Page**  
  포트폴리오 소개 및 주요 프로젝트 요약

- **Projects Page**  
  개인 / 회사 프로젝트 탭 전환 기능 구현

- **Project Detail Page**  
  프로젝트 상세 설명 페이지

- **About Page**  
  기술 스택 및 자기소개

## 📌 Key Features

- Dynamic Route(`[id]`)를 활용한 프로젝트 상세 페이지 구현
- 반응형 레이아웃 구현
